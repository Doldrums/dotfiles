###### install

```bash
git clone https://github.com/Doldrums/dotfiles.git ~/dotfiles
cd ~/dotfiles && ./install.sh
```

###### structure

```
zsh/        shell config
starship/   prompt
ghostty/    terminal + moonfly theme
nvim/       kickstart custom plugins
```
